API Nómina

Esta es una API REST para gestionar nóminas. Puedes realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre las nóminas.

Para interactuar con la API, selecciona un servidor y utiliza los siguientes endpoints:

Los métodos disponibles son:

GET
/nomina
Obtener todas las nóminas
Descripción: Devuelve un listado de todas las nóminas registradas.
Respuesta: 200, Devuele un JSON con array de nóminas.
▼
GET
/formulario
Crear una nueva nómina
Descripción: Crea una nómina nueva.
Body: Devuelve un HTML con los datos de la nómina ejemplo:
Respuesta: Devuelve un String de objeto nómina creada.
▼
GET
/nomina/{id}
Obtener nómina por ID
Descripción: Devuelve la nómina correspondiente al ID del JSON.

{ "id_empleado": 1, "nombre_empleado": "Juan Pérez", "salario": 2500.0, "dias_trabajados": 30 }

Respuesta: 200 OK, ID JSON nómina.
▼
GET
/formulario-actualizar/{id}
Actualizar una nómina
Descripción: Actualiza los datos de una nómina existente.
Body: Un HTML con los nuevos datos.

{ "id_empleado": 1, "nombre_empleado": "Juan Pérez", "salario": 2500.0, "dias_trabajados": 30 }

Respuesta: Devuevle un String de objeto nómina actualizada.
▼
POST
/nomina
Crea una nueva nomina
Descripción: Crea una nueva nomina en el JSON
Body:

{ "id_empleado": 1, "nombre_empleado": "Juan Pérez", "salario": 2500.0, "dias_trabajados": 30 }

Respuesta: 201 OK. Devuelve un JSON
▼
PUT
/nomina/{id}
Actualiza nomina
Descripción: Actualiza una nomina ya existente en el JSON a traves de su ID.
Body:

{ "id_empleado": 1, "nombre_empleado": "Juan Pérez", "salario": 2500.0, "dias_trabajados": 15 }

Respuesta: 200 OK. Devuelve un JSON.
▼
GET
/htmlnomina
Vista de la tabla nomina
Descripción:
Body: Devuelve un HTML con los datos de la nómina
Respuesta: Devuelve un string con los datos que contiene la tabla nomina.
▼
DELETE
/nomina/{id}
Eliminar una nómina
Descripción: Elimina la nómina con el ID indicado del JSON.
Respuesta: 204 No Content.
▼
